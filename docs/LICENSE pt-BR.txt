* Detalhes: Utilizada para fazer a ponte de baixo nível via API Vulkan na GPU 
  e possibilitar a injeção síncrona de shaders na pipeline de vídeo.

3. ALGORITMO DE ESCALA ESPACIAL: AMD FidelityFX™ Super Resolution (FSR)

* Licença Aplicável: Licença MIT (Open-Source, permissiva).
* Detalhes: Tecnologia integrada no bloco lógico do shader para reconstrução de

--------------------------------------------------------------------------------
4. ALGORITMO DE CONTROLE TEMPORAL: INTERSECTION FLUID SHARPEN (IFS)
--------------------------------------------------------------------------------
* Versão Utilizada: IFS v1.0
