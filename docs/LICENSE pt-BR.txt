Este pacote standalone distribui uma pipeline de pós-processamento modular 
composta por ferramentas de código aberto integradas a uma lógica autoral.

--------------------------------------------------------------------------------
2. BIBLIOTECA DE RENDERIZAÇÃO E INTERPOLAÇÃO (IFS): libplacebo
--------------------------------------------------------------------------------
* Versão Utilizada: Compilação integrada nativamente no binário FFmpeg core acima.
* Copyright: Copyright (c) 2016-2026 Niklas Haas and libplacebo contributors
* Licença Aplicável: GNU Lesser General Public License (LGPL v2.1 ou posterior).
* Detalhes: Utilizada para fazer a ponte de baixo nível via API Vulkan na GPU 
  e possibilitar a injeção síncrona de shaders. É o motor real responsável por 
  dar vida ao conceito IFS (Interpolated Frame Sampling) através de seus algoritmos 
  nativos de amostragem e mesclagem temporal de quadros.

--------------------------------------------------------------------------------
3. ALGORITMO DE ESCALA ESPACIAL: AMD FidelityFX™ Super Resolution (FSR)
--------------------------------------------------------------------------------
* Versão Utilizada: FSR v1.0.2
* Desenvolvido por: AMD (Advanced Micro Devices, Inc.)
* Licença Aplicável: Licença MIT (Open-Source, permissiva).
* Detalhes: Tecnologia integrada no bloco lógico do shader (portabilidade GLSL 
  por agyild) para reconstruction de bordas físicas e redimensionamento de alta 
  fidelidade sem uso de IA.

--------------------------------------------------------------------------------
4. SCRIPTS DE INTEGRAÇÃO DA PIPELINE: FSRIFS
--------------------------------------------------------------------------------
* Versão Utilizada: v1.1.0 (Arquitetura Modular)
