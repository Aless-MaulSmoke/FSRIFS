# Mandatory FFmpeg Architecture

To run this pipeline from the source code, you must maintain the official directory structure and place the specific compiled executable inside the `bin` folder.

### Required Version:
* **Build:** ffmpeg-master-build-N-125258-gdf94900c98-20260624 (compiled with libplacebo and Vulkan support)

### Target Path:
`ffmpeg/bin/ffmpeg.exe`

Ensure the executable is placed in the `bin` subdirectory before executing the scripts.
